# UnnesHub Design Guidelines

This document outlines the design system and UI rules for UnnesHub to ensure visual consistency across the platform.
create mobile first design. always put mobile as the priority and desktop as the secondary.

## Color Palette

### Base Colors

- **Primary Navy:** `#0B1E36` (Used for main backgrounds, headers, and dark text)
- **Accent Yellow:** `#F2C010` (Used for primary buttons, active states, and highlights)
- **Pure White:** `#FFFFFF` (Used for body backgrounds and light text)

### Neutrals & Component Colors

- **Surface Gray:** `#E2E5E9` (Used for card containers and registration form background)
- **Input Field Gray:** `#8FA0AF` (Used for dark-themed text inputs)
